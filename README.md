# super-kamusi
This repo aims at coming up with a Correct Swahili-English dictionary being based in Kiswahili. 
JSON file consists of over 10000+ words.

This repo is an upgrade from [Kalebu-kamusi](https://github.com/Kalebu/kamusi/), Thanks to great effort done by [Kalebu](https://github.com/Kalebu).
